Mapbox API Access Token

How to add a Mapbox API Access Token to MapHub

  1. Sign up to for free account on Mapbox: https://account.mapbox.com/auth/signup/

  2. Navigate to the Tokens section in your account:

  3. Under access tokens, look for Default public token, and click the blue icon to copy the token to the clipboard. You can also select the text and copy it manually. Important: make sure the token you are selecting starts with pk.

  4. Go to your MapHub account's Basemaps Settings: https://maphub.net/settings/basemaps

  5. Paste the copied text to Mapbox Access Token field and click Save.

  6. Congratulations, now you have unlimited basemap usage for Mapbox on MapHub.
