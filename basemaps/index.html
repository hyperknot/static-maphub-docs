Basemaps

MapHub provides a wide range of basemaps, some of which are from commercial providers.

We offer all maps for free, but if you are one of our heaviest users, we ask you to add a free API key to your account.

Without an API key, maps with heavy usage will be limited to MapHub Classic for 24 hours.
