How to add a Thunderforest API key to MapHub

  1. Sign up to a free Hobby Project account on Thunderforest Maps: https://www.thunderforest.com/pricing/
  2. Confirm your account via the verification email.
  3. Log in with your registered credentials.
  4. Copy the API Key from the dashboard.
  5. Go to your MapHub account's Basemaps Settings: https://maphub.net/settings/basemaps
  6. Paste the copied text to Thunderforest API key field and click Save.
  7. Congratulations, now you have unlimited basemap usage for Thunderforest on MapHub.
